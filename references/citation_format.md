# CBETA 引用格式说明

## 标准引用格式

```
《经名》：「引用文」(CBETA 版本, 册号, no. 经号, p. 页码栏行)
```

---

## 格式详解

### 1. 经名

使用佛典完整题名（繁体）：
- `金刚般若波罗蜜经` → `金剛般若波羅蜜經`
- `妙法莲华经` → `妙法蓮華經`

---

### 2. 版本

当前最新版本：`CBETA 2025.R3`

版本号组成：
- `2025` - 年份
- `R3` - 第3次发布

---

### 3. 册号

格式：`T08` (藏经代码 + 册号)

藏经代码：
| 代码 | 名称 |
|------|------|
| T | 大正藏 |
| X | 卍续藏 |
| N | 南传大藏经 |
| K | 高丽藏 |

---

### 4. 经号

格式：`no. 235` 或 `no. 0235`

完整经号如 `T08n0235`：
- T08 = 大正藏第8册
- n0235 = 第235号经典

---

### 5. 页码栏行

#### 页码格式

**单行**:
```
p. 749c22
```

**跨行（同栏）**:
```
p. 749c22-23
```

**跨栏**:
```
p. 749c22-b05
```

**跨页**:
```
pp. 749c22-750a05
```

#### 栏位代码

| 代码 | 栏位 |
|------|------|
| a | 上栏 |
| b | 中栏 |
| c | 下栏 |

传统佛经排版为每页三栏，每栏约28行。

---

## Linehead 解析

**格式**: `T08n0235_p0749c22`

**解析**:
| 部分 | 含义 |
|------|------|
| T | 藏经代码（大正藏） |
| 08 | 册号 |
| n0235 | 经号 |
| p0749 | 页码 |
| c | 栏位（下栏） |
| 22 | 行号 |

## 论文引用格式

### APA 格式

```
(CBETA, 2025.R3, T08, no. 235, p. 749c22-23)
```

### 注脚格式

```
CBETA 中华电子佛典协会. 《金刚般若波罗蜜经》. CBETA 2025.R3, T08, no. 235, p. 749c22-23.
```

### 参考书目格式

```
CBETA 中华电子佛典协会. 《金刚般若波罗蜜经》. CBETA 2025.R3.
https://api.cbetaonline.cn
```

## 特殊情况

### 1. 跨多行

使用 `pp.` 前缀：
```
pp. 749c22-750a05
```

### 2. 藏经代码变体

某些论文使用变体：
- `《大正藏》册8，第235号` → 对应 `T08, no. 235`
- `R130` → 卍续藏编号

### 3. 南传大藏经

```
《相應部》：「...」(CBETA 2025.R3, N13, no. 6, p. 1a12)
```
